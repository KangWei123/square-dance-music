# 广场舞曲播放器微信小程序

## 项目简介
专为中老年广场舞群体设计的轻量化微信小程序，主打「免费、操作简单、音源全、无广告干扰」。

## 核心功能
- 基础播放：按舞种、节奏、热度分类，支持单曲播放/循环/随机、倍速调节、音量独立调节
- 收藏功能：一键收藏常用曲目，生成「我的广场舞歌单」，支持歌单重命名、删除
- 离线缓存：免费缓存5首热门曲目，会员解锁无限缓存
- 操作优化：大尺寸按钮、大字体、语音点歌
- 辅助功能：显示曲目时长，支持后台播放

## 技术实现
- 前端：微信小程序原生开发（WXML/WXSS/JS）
- 后端：微信云开发（云数据库+云存储+云函数）

## 项目结构
```
square-dance-music/
├── pages/
│   ├── index/         # 首页（分类和音乐列表）
│   ├── player/        # 播放器页面
│   ├── collect/       # 收藏页面
│   └── cache/         # 缓存页面
├── images/            # 图标目录
├── app.js             # 小程序入口
├── app.json           # 小程序配置
├── app.wxss           # 全局样式
├── package.json       # 项目配置
└── sitemap.json       # 小程序索引配置
```

## 开发说明
1. 确保已安装微信开发者工具
2. 在微信开发者工具中导入此项目
3. 配置云开发环境ID（在app.js中修改）
4. 创建必要的数据库表结构
5. 上传音乐文件到云存储

## 注意事项
- 图标文件需要用户自己添加到images目录
- 项目使用模拟数据，实际使用时需要替换为真实数据
- 语音点歌功能需要接入语音识别API
- 会员功能需要接入支付API

## 上线流程
1. 完成一期开发（核心播放+分类+收藏+基础缓存）
2. 上线测试用户体验
3. 完成二期开发（广告入口+会员功能+语音点歌）
4. 正式推广

## 变现方式
- 底部横幅轻广告
- 会员订阅
- 本地商家引流

## 目标用户
50-70岁广场舞爱好者，子女可协助注册使用。

## 推广方式
- 线下广场舞团队扫码推广
- 中老年社群/公众号转发
- 微信附近的小程序推荐
